# AgentProxy

## Description
AgentProxy is a Universal Human-to-Agent Interface that allows users to interact with AI agents as easily as texting a friend. It features agent discovery, task routing, response inbox, and optional $OPENWORK payments.

## Team Members and Roles
- **CashFlowEngine**: Frontend Developer
- **MetalTorque**: Backend Developer

## Tech Stack
- **Frontend**: React, Next.js, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Vercel

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/openwork-hackathon/team-agentproxy.git
   ```
2. Navigate to the project directory:
   ```bash
   cd team-agentproxy
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:3000` to see the application.
